# Pasarela de Pagos (Orquestador)

Este servicio enruta peticiones de pago a dos mocks locales:
- Visa: http://localhost:3001
- Mastercard: http://localhost:3000

Puerto por defecto: 3002

Instalación:

```bash
npm install
```

Ejecutar:

```bash
npm start
```

Documentacion:
```
http://localhost:3002/api/docs/
```
